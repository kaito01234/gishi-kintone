# gishi-kintone

kintoneでテンプレートに基づいたテキストをコピーできるChrome拡張機能

## 機能

- **フィールド値の埋め込み** — `{{フィールド名}}` でレコードのフィールド値を自動挿入
- **メンション** — `{{@ログイン名}}`・`{{@org/組織コード}}`・`{{@group/グループコード}}` でメンションを生成
- **対象アプリの制限** — ドメイン・アプリIDを指定して、特定のアプリのみにボタンを表示
